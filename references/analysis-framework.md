                                13. ## 总体原则
                               
                                14. ### 证据链思维
                               
                                15. 每个推断都应该有清晰的证据链：
                                16. ```
                                    观察到的信号 → 推理逻辑 → 推断结论 → 置信度评估
                                    ```

                                    ### 置信度评估标准

                                    | 置信度 | 标准 | 典型场景 |
                                    |--------|------|----------|
                                    | **高** | 多条独立证据互相印证 | 用户自述 + 多处线索一致 |
                                    | **中** | 有明确信号但存在替代解释 | 品牌提及推断消费力 |
                                    | **低** | 信号微弱或高度间接 | 仅从话题偏好推断教育水平 |

                                    ### 交叉验证

                                    同一结论如果从多个维度独立得到，置信度上调。

                                    ### 三层证据体系

                                    对于理想型推断等高度推断性的维度，使用三层证据：
                                    1. **发布内容证据**（前台表演）：TA 主动展示给别人看的
                                    2. 2. **收藏行为证据**（内隐偏好）：TA 私下关注但不一定公开的
                                       3. 3. **理论依据**：心理学研究支撑推断的合理性
                                         
                                          4. 三层证据一致时，推断置信度最高。仅有单层证据时，必须标注为「弱推断」。

                                          14. | 自我指称线索 | "刚毕业的" / "带娃第三年" | 直接线索，置信度高 |
                                         
                                          15. ### 性别
                                         
                                          16. 从昵称、自拍、话题领域判断。不要刻板印象化。无法确定就标注「无法确定」。

                                          ### 地理位置

                                          强信号：IP属地、频繁出现的地名/商圈、方言词汇
                                          弱信号：偶尔提到某城市（可能旅游）

                                          ### 职业

                                          信号：工作日/周末发布差异、专业术语、工作场景描述、下班后活动

                                          ### 城市线级

                                          结合地理位置 + 消费场所 + 品牌选择综合判断。

                                          ## 维度二：消费与生活方式

                                          ### 消费能力等级

                                          | 等级 | 特征 |
                                          |------|------|
                                          | **高消费** | 奢侈品牌、高端餐饮、商务舱 |
                                          | **中高消费** | 轻奢品牌、网红餐厅、精品酒店 |
                                          | **中等消费** | 快时尚、大众消费场所 |
                                          | **性价比导向** | 平替推荐、折扣信息 |
                                          | **学生/入门** | 校园场景、实习相关 |

                                          注意区分「自己消费」和「测评/种草」。

                                          ### 兴趣标签提取

                                          从笔记主题聚类提取 8-15 个兴趣标签，按频率分三级：
                                          - 核心兴趣（>30% 笔记）
                                          - 次要兴趣（10-30%）
                                          - - 偶发兴趣（<10%）
                                           
                                            - ### 收藏内容 vs 发布内容的差异

                                            这是画像分析中最有洞察力的部分。重点分析「收藏了但没发过」的主题——这代表 TA 的隐藏兴趣：可能是 TA 想做但还没做的事、想去但还没去的地方、或者不好意思公开分享的偏好。

                                            这些隐藏兴趣对追求策略价值极大——它们是「还没被别人看到的共鸣点」。

  ## 维度分析框架

  ### 维度一：基础人口学推断
  - 年龄段：网络用语、话题偏好、消费场景、文化引用
  - 性别：昵称、自拍、话题领域
  - - 地理位置：IP属地、地名/商圈、方言词汇
    - - 职业：工作日/周末发布差异、专业术语、工作场景
    - 城市线级：地理 + 消费场所 + 品牌选择
    -
    - ### 维度二：消费与生活方式
    - - 消费能力：高消费、中高、中等、性价比、学生入门
    - 兴趣标签：核心（>30%）、次要（10-30%）、偶发（<10%）
    - - 收藏差异：发布内容 vs 收藏内容的洞察力最大——隐藏兴趣代表未被看到的共鸣点
      - 
      ### 维度三：心理与人格特征
      Big Five人格推断：
      - 开放性(O)：探索新事物、旅行、创意 vs 重复性内容
      - - 尽责性(C)：详细攻略、清单、计划性 vs 随性记录
      - 外向性(E)：社交场景多、互动积极 vs 独处场景
      - - 宜人性(A)：正面评价、利他分享 vs 批评、锐评
        - - 情绪稳定性(N)：情绪波动 vs 平稳理性
          -
          - 沟通风格：文案长度、emoji使用、语气特征、英文夹杂、标点习惯、叙事视角
          
          ### 维度四：社会关系与影响力
          - 头部KOL(>50万)：评论区脱颖而出
          - 腰部KOL(5-50万)：可能有助理管理
          - KOC(1-5万)：亲力亲为，回复率高
          - - 活跃用户(1000-1万)：最容易建立连接
          - 普通用户(<1000)：每条互动都被看到
          -
          - ### 维度五：时间行为模式
          - - 工作日晚间密集 → 上班族
            - - 全天不规律 → 自由职业/学生/博主
              - - 周末爆发 → 周末才有时间的上班族
                - - 深夜频繁 → 夜猫子
                  - - 主题演变：兴趣转移、生活阶段变化、风格变化
                    - 
                    ### 维度六：社交互动偏好
                    评论区回复分析：
                    - 提问型：回答率和风格
                    - - 共鸣型：热情互动程度
                      - - 专业讨论型：深度讨论意愿
                      - 幽默调侃型：接梗能力
                      - 纯夸赞型：反应方式
                      - 
                      ### 维度七：情感状态推断
                      **谨慎推断** - 准确率极低，需声明仅供参考
                      - 照片中重复人物：可能伴侣（也可能朋友家人）
                      - - 复数表述：可能伴侣（也可能朋友）
                      - 单身宣言：目前单身（可能是玩梗）
                      - 情感类收藏：有情感需求（不确定什么需求）
                      
                      ### 维度八：「吸引力密码」综合推断
                      1. TA最引以为豪的身份/经历
                      2. TA最在意的生活品质元素
                      3. TA的社交需求类型（展示型/交流型/记录型）
                      4. 什么类型的人最可能引起兴趣
                      5. 「前台」vs「后台」差异——收藏内容、低互动笔记、深夜碎碎念更接近真实
