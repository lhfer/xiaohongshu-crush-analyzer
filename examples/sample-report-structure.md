# 报告结构示例

生成的 HTML 报告包含 7 个 Tab，以下是每个 Tab 的内容概览。

## Tab 1: 📊 总览

- 一句话用户画像
- 核心标签云（8-15个）
- 基础数据展示（粉丝/关注/获赞与收藏）
- 五维雷达图（Chart.js）：
  - 开放性 (Openness)
  - 独立性 (Independence)
  - 浪漫感性 (Romanticism)
  - 社交影响力 (Social Influence)
  - 生活品质 (Lifestyle Quality)

## Tab 2: 🔍 深度画像

8 个分析维度，每个维度包含：
- 分析结论
- 证据来源（具体笔记引用）
- 置信度标注（高/中/低）

维度列表：
1. 基础人口学
2. 消费与生活方式
3. 心理与人格（含 Big Five 柱状图）
4. 社会关系与影响力
5. 时间行为模式
6. 社交互动偏好
7. 情感状态推断
8. 「吸引力密码」

## Tab 3: 💜 理想型画像

- 5 大心理学理论框架卡片
- 六维度理想型雷达图
- 每个维度的详细分析（证据 → 理论 → 结论）
- **交互式匹配度自测工具**（滑块 + 实时计算匹配百分比）

## Tab 4: 📋 笔记一览

- 全部笔记的可搜索列表
- 分类筛选按钮（旅行/成长/音乐/美食/...）
- 按点赞数排序
- 笔记类别分布统计

## Tab 5: 🎯 追求策略

- 核心策略原则
- Do's & Don'ts 清单
- 五阶段路线图（每阶段含具体行动 + 分支应对）

## Tab 6: 💬 话术锦囊

- 评论话术模板（按笔记类别分类）
- 私信开场白（2-3个版本，不同切入角度）
- 话题推进指南
- 转微信话术

## Tab 7: ℹ️ 方法论

- 数据采集报告
- 分析方法说明
- 心理学理论引用
- 局限性声明
- 伦理声明

---

## 技术规格

- 单文件 HTML，全部内联 CSS/JS
- Chart.js via CDN
- 主色调 #FF2442（小红书品牌色）
- 响应式布局
- 置信度颜色系统：高=#22c55e / 中=#eab308 / 低=#9ca3af
